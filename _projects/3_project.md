Online educational videos generate massive streams of learner-generated questions, but little is known about how question complexity relates to engagement and learning outcomes. **YouLeQD** is a dataset and framework for analyzing the cognitive complexity of questions posed by learners in platforms like YouTube.

We annotate questions using Bloom's Taxonomy cognitive levels and build classifiers to automatically predict question complexity. Analysis reveals connections between question complexity distribution and viewer engagement patterns, providing actionable insights for course designers.

### Key Contributions

- YouLeQD: a large-scale dataset of learner questions with cognitive complexity labels
- Transformer-based question complexity classifiers
- Engagement analysis linking question complexity to view and interaction metrics
