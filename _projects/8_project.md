High-quality classroom discourse — where students engage in knowledge construction rather than mere recall — is a strong predictor of learning outcomes. Yet teachers receive little real-time support for recognizing and promoting productive reasoning patterns. **ADAS (Automated Discourse Analysis System)** addresses this gap.

Using **LLM-based data augmentation** and **fine-tuned transformer models**, ADAS classifies reasoning components and utterance types in transcribed science classroom dialogue. The system analyzes how cognitive complexity evolves over a lesson and identifies the instructional moves associated with higher-order reasoning, giving teachers evidence-based feedback to improve discourse quality.

This project is funded by the **Interdisciplinary Seed Grant, Kennesaw State University**.

### Key Contributions

- Automated classification of reasoning utterances in science classroom transcripts
- LLM-augmented training data pipeline to address low-resource discourse annotation
- Teacher-facing dashboard for visualizing discourse patterns across lessons
